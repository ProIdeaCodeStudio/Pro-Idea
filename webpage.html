A tag together with its children is an element. The children are the words inside the tag. The head tag is a tag that contains some information necessary for the browser.
The div tag is an html element that is used to group other html elements together. It stands for division and is used to divide or group content into logical sections.
It's a block-level element, meaning it takes up the whole width of its container and creates a break after it.

The img tag is used for images. The src attribute is used to specify the url pr file path of the image.
The alt attribute is for givng the image an alternative image for screen readers or if the image isn't loading. The width and height attributes can be used to specify the size of the image in pixels or as a precentage of the containing element.
The img tag is a self-closing tag.
